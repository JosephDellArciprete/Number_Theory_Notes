    Section 1.1 Sets

    Subsection 1.1.1 Sets and Subsets

    Sets are determined uniquely by their elements. The primitive, undefined concept in set theory is that of membership.
    🔗

    Definition 1.1.1.

    A set is a collection of objects, called the elements of the set. To denote an element \(x\) is a member of a set \(A\) we write \(x \in A\text{.}\) If \(x\) is not an element of \(A\) we write \(x \not\in A\text{.}\) By definition a set is totally determined by its elements, so two sets are equal if and only if they have the same elements:
    \begin{equation*} A = B \iff \forall x (x \in A \iff x \in B)\text{.} \end{equation*}
    🔗
    🔗
    As mentioned above the key concept is that of membership, we do not care about repeats of element nor of the order of the elements. The set of US presidents between 1789 to 2026 is the same if we list the in ascending order, descending order, or if we repeat some of the presidents in our description of the set
    🔗
    We immediately come to another simple concept for out next definition.
    🔗

    Definition 1.1.2.

    If A and B are sets we say that A is a subset of B, written \(A \subseteq B\text{,}\) if every element of A is also an element of B, that is:
    \begin{equation*} A \subseteq B \iff \forall x (x \in A \implies x \in B) \end{equation*}
    🔗
    🔗
    We must not abuse the the concepts of set membership too much though, consider Russel’s Paradox, named after logician Bertrand Russell.
    🔗

    Warning 1.1.3. Russell’s Paradox.

    Let \(R\) be the set of all sets \(A\) such that \(A \not\in A\text{.}\) Is \(R \in R\text{?}\) One the one hand if \(R \in R\) then by definition, \(R \not\in R\text{.}\) However if \(R \not\in R\) again by definition \(R \in R\text{.}\) So in either case we get a contradiction. Hence R is not a well defined set.
    🔗
    🔗
    One way to avoid such problems is to have a well defined universal set \(U\) and define a subsets of this set by defining a valid property that defines a subset. Particular for a set \(U\) and a predicate \(p(x)\) on \(U\text{,}\) i.e a valid statement that is true of false, then:
    \begin{equation*} \{ x \in U : p(x) \} \end{equation*}
    Is a subset on \(U\text{,}\) this is known as predicate form.
    🔗
    Another easy to define a set is simply to list its elements, straightforward but obviously not often feasible.
    \begin{equation*} \{ \text{element 1}, \text{element 2}, \ldots, \text{last element} \} \end{equation*}
    🔗
    On the other hand we also have the empty set, denoted \(\emptyset\text{,}\) the set containing no elements.
    🔗
    🔗
